#EMPLOYEE PERFORMANCE DASHBOARD
▪	Designed an interactive Employee Workforce Dashboard using MS Excel to analyse active and inactive employee data.
▪	Cleaned and structured raw employee data including EEID, Gender, Age, Department, Job Role, Hire Date, Exit Date, Salary, and Status.
▪	Created custom age group buckets (26–30, 31–35, 36–40, 41–45, 46–50, 51–60) using Excel formulas for workforce segmentation.
▪	Built Pivot Tables to analyse employee count by Status, Gender, Department, Business Unit, and Age Group.
▪	Used GETPIVOTDATA function to dynamically fetch KPI values and ensure accurate dashboard reporting.
▪	Calculated key HR metrics such as total headcount, active employees, inactive employees, and attrition trends. 
▪	Developed visualizations including bar charts and summary tables for quick management insights.
▪	Automated dashboard updates by linking pivots and formulas, reducing manual reporting effort.
▪	Improved decision-making support by presenting data in a clear, structured, and management-friendly format.
